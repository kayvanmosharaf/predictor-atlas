## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Amplify Sandbox

Start the local Amplify backend:

```bash
npx ampx sandbox
```

## Seed the Database

With the sandbox running, seed predictions and outcomes:

```bash
npx tsx scripts/seed.ts
```
